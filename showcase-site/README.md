# OpenGed Showcase (repo séparé)

Ce dossier contient une version **standalone** du site vitrine, prévue pour être déplacée dans un dépôt Git séparé.

## Démarrage local

Comme c'est un site statique, ouvrez simplement `index.html` dans un navigateur.

## Publication recommandée

- GitHub Pages
- Netlify
- Vercel (mode static)

## Créer le repo séparé

1. Créez un nouveau dépôt, par ex. `openged-showcase`.
2. Copiez le contenu de ce dossier dans la racine du nouveau dépôt.
3. Poussez sur GitHub.
4. Activez la publication (Pages/Netlify/Vercel).

## Contenu inclus

- landing page française orientée produit open source
- sections: promesse, bénéfices, workflow, cas d'usage
- aucun bloc pricing (conforme à la stratégie actuelle 100% open source)
